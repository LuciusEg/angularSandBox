#Interceptors
Меняет запрос (дополнительные параметры/хедеры)
Например токен авторизации
Можем посмотреть что приходит с сервера и обработать общую ошибку один раз для всех запросов 

HttpClient -> interceptor -> API
HttpClient <- interceptor <- API
